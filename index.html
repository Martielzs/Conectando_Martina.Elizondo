¿QUÉ ES
CONECTANDO?

Conectando es un juego de construcción y exploración en el que los jugadores deben encontrar una salida a través de un sistema que se transforma constantemente.

Utilizando distintas piezas, el tablero se construye progresivamente, generando recorridos, conexiones y nuevas posibilidades de desplazamiento a medida que avanza la partida.

Cada decisión modifica la configuración existente, haciendo que el camino hacia la salida cambie continuamente.

Construir
para avanzar

La partida comienza desde un punto central del tablero hacia las metas ubicadas en las esquinas. Si al completar una trayectoria los equipos descubren que la salida elegida no es la correcta, las estructuras construidas se mantienen: el juego exige transformar el sistema y generar una nueva ramificación continuando desde el camino previo para alcanzar el objetivo válido.

● Configuración A: Trayectoria Inicial

Ambos equipos trazan sus recorridos originales hacia las salidas inferiores, fijando las primeras estructuras en el tablero.

● Configuración B: Adaptación y Ramificación

Al comprobar que las salidas inferiores no son válidas, los equipos no desarman lo construido: extienden y ramifican nuevas conexiones desde los extremos alcanzados hacia las metas superiores.

FUNDAMENTO

Este proyecto abstrae el juego físico materializando sus conceptos directamente en el tablero. El entorno no viene predefinido, sino que se construye gradualmente con las acciones del jugador.

Las piezas aisladas no significan nada; es su conexión interactiva la que altera el estado del sistema, revelando caminos ocultos y transformando el trayecto en una estructura viva, dinámica y progresiva.

// EXPLORACIÓN

Es el proceso de descubrir cómo se forman los caminos: el jugador reconoce nuevas posibilidades al interactuar con las piezas y ampliar el recorrido.

// TRANSFORMACIÓN

Es el cambio progresivo del sistema: los componentes se conectan y generan nuevas configuraciones, modificando la estructura constantemente.

// INCERTIDUMBRE

Aparece al no poder prever el mapa final: cada decisión abre y cierra caminos de forma simultánea, volviendo inestable el recorrido.

// CONFIGURACIÓN

Es la organización temporal del sistema: el resultado momentáneo de cómo se conectan las piezas, creando un tablero distinto en cada partida.
